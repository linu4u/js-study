#### \*\* javascript 코드 \*\*

>ver 1
>
> ```js
> var srcElem = document.getElementsByTagName("li");
var liLen = srcElem.length;
>
> for ( i = 0; i < liLen; i++ ) {
	if ( i == 0 || i % 3 === 0 ) {
		srcElem[i].className += 'color_red';
	} else if ( i % 3 === 2 ) {
		srcElem[i].className += 'color_blue';
	}
} 
> ```
